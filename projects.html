Same Level Prototype:

Description: This is a project made early 2021 when I was learning some new skills in game development. The game is simply the same level but with many twists. What this entails is the first level repeating over and over but each time a new change happens. This change can range from inverted controls, player sizes, reduce visibility and many more. The game is perfect for speedrunning but can also be played casually.
